# AI Job Application Copilot

A full-stack SaaS application that helps software engineers track job applications, prepare for interviews, and optimize their job search using analytics and AI.

This project is being built as a production-style learning project to practice:

- Backend API development (FastAPI)
- Frontend development (Next.js)
- PostgreSQL database design
- Async workers (Redis + Celery)
- Docker & AWS deployment

## Tech Stack

Frontend:

- Next.js (TypeScript)

Backend:

- FastAPI
- PostgreSQL
- Redis + Celery

Infrastructure:

- Docker (local)
- AWS ECS + RDS (planned)

See `docs/` for product, architecture and roadmap.
